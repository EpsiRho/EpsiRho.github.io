Themes

I have recently been using Zen Browser as my main web browser, a firefox based browser that's got lots of nice QOL. I stumbled upon the fact that Zen supports Transparency by setting the browser.tabs.allowtransparentbrowser tag in about:config. This disables the background between the webpage compositor and the app background, and alongside MicaForEveryone on windows (I hear Linux has easier support) you can allow a webpage to be fully transparent back to the desktop. This makes for some very nice looks and lots of customization. Unfortunately, there is no way to detect if this is enabled, so I needed to add a Theme Switcher. Alongside that I figured I might as well make a few themes to flesh out the menu xP


The current themes are:

  • Notebook (Warm White)
  • Flashbang (Cool White)
  • Dark (Dark Gray)
  • Transparent (Transparent)
  • Ocean (Deep cyan and blue)
  • Dream (Purple and Pink)
  • Fields (Green)
  • Sunset (Red and Purple)
  • Matrix (Black and Green)

If you have suggestions for more themes or fixes/adjustments to current ones, please leave a Github Issue/PR :D


Comments

I saw a post awhile back from a Bluesky user turning their posts into repos for their blog's comments. I've been meaning to look into it since then, and now I've lost that post hah! But I did finally get around to it, and now blog posts can have comments! This one will have them, you should be able to see them below and going to the Bluesky post will allow you to leave your own comment. You totally should btw, y'know, so I can test my code :P

The implementation is custom, so not easy to drag n drop into other code. But feel free to look at mine for reference. The simple idea is we call app.bsky.feed.searchPosts to get all the posts I have made tagged "#blog", and checking if it's link is the same as the one we're on. If it is, then we call app.bsky.feed.getPostThread to get all of it's replies. Then we can recursively follow the reply tree to generate HTML to show.
